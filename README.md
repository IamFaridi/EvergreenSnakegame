# EvergrenSnakegame
Snake is the common name for a video game concept where the player maneuvers a line which grows in length, with the line itself being a primary obstacle.
<br>
Instruction to play!!
<br>
use "a" to go "LEFT"
<br>
use "s" to go "RIGHT"
<br>
use "z" to go "DOWN"
<br>
use "w" to go "UP"
<br>
use "x" to cancel game anytime
